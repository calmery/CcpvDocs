チュートリアル

このチュートリアルでは CSS（Cascading Style Sheets）の基礎文法を説明します．

CSS とは？

HTML の見た目の装飾を行うために必要となるものです．

どこで使うのか？

Web サイトの見た目の装飾を指定するために使用されています．例えば文字色や文字サイズ，背景色などです．
